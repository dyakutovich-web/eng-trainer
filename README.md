# English Vocab Trainer

> Персональный тренажёр английской лексики. Это файл-ТЗ: что строим, как работает, ключевые решения.
> Дорожная карта и этапы — в `TASK.md`. Модель усвоения — **v2** (баллы + два этапа).

## Что это
PWA (веб, телефон + десктоп, офлайн) для расширения активного словарного запаса английского у одного
пользователя (Денис). Учим **самые частотные** слова и выражения **в контексте**, с проверкой по
модальностям. Ядро — система усвоения со списками слов, баллами по типам упражнений и двумя этапами
(быстрый + 30-дневная консолидация).

## 1. Пользователь и платформа
- Один пользователь → без авторизации, данные локально (localStorage).
- **PWA**, офлайн, ставится на телефон.
- **Озвучка (TTS)** — браузерный Web Speech (бесплатно). **Распознавание (ASR)** — Web Speech с fallback
  на самооценку (iPhone). В аудировании и произношении есть кнопка **«Пропустить — без очков»**.

## 2. Контент
- База: **15 000 частотных лемм** (`data/wordbase.csv`), POS-разметка, ввод по частоте.
- Выражения: ~150 phrasal verbs + ~400 коллокаций + ~150 идиом.
- На каждое слово: **значения `senses[]`** (2–4 самых частотных, у каждого свой перевод RU, часть речи,
  пример), IPA, usage notes. Для **неправильных глаголов** — формы `{past, pp}`.
- Картинки — опционально, только для конкретных существительных/глаголов.
- Генерация контента — AI + TTS, траншами по частоте.

## 3. Списки слов (слово всегда ровно в одном)
- **Каталог** — все слова, ещё не тронуты.
- **Активные (учу)** — Этап 1 или Этап 2.
- **Железно знаю** — полностью вне тренировок (можно вернуть).
- **Выучено** — прошли оба этапа; отдельный список, можно тестировать и возвращать.

## 4. Пополнение (отдельная кнопка «Пополнить слова»)
Слова НЕ влетают в учёбу сами. Пользователь заходит в каталог (фильтр по части речи), по каждому слову:
- **Не знаю** → Активные, Этап 1
- **Знаю → показать значение** → раскрываю все значения → выбор: **железно** / **пассивно** / **не знаю**
  - железно → «Железно знаю»
  - пассивно → Активные, **Этап 2** (особые условия)
  - не знаю → Активные, Этап 1
- **Железно знаю** → «Железно знаю» (без тренировок)
- **Пропустить** — остаётся в каталоге
Прерывается в любой момент.

## 5. Модальности (типы упражнений = оси баллов)
| Тип | Что проверяет |
|---|---|
| **Аудирование** | слышу TTS → выбираю слово (есть «Пропустить») |
| **Перевод** | вижу слово → выбираю значение; драятся **все** senses (включая вторичные) |
| **Контекст** | вставить слово в предложение |
| **Написание** | пишу слово по-английски из перевода |
| **Произношение** | говорю → ASR / самооценка (есть «Пропустить») |
| **Формы** | только irregular verbs: ввод Past Simple (V2) и Past Participle (V3) |
Выражения — без произношения.

## 6. Система усвоения (v2)
Баллы начисляются **по каждому типу упражнения отдельно**: **+1** за верно, **−2** за неверно (не ниже 0).
Прогресс-бары по типам видны во время тренировки и обновляются сразу.

- **Этап 1 (быстрый):** нужно **10** баллов по каждому типу. Тип, набравший 10, для этого слова закрывается
  (остаются недобранные). Все типы по 10 → переход на Этап 2 (баллы сбрасываются).
- **Этап 2 (консолидация):** нужно **30** баллов по каждому типу, каждый тип **не чаще 1 раза в день**
  → минимум ~30 дней. Все типы по 30 → слово уходит в «Выучено».
- **Пассивные** слова стартуют сразу в Этапе 2 с нуля.
- Пропуск аудирования/произношения — очки не меняются.

Прогрессивная разблокировка как жёсткий замок — убрана: все типы доступны сразу, сложность держит штраф −2,
а порядок (рецепция → продукция) задаёт мягкий приоритет показа.

## 7. Тренировка и тест — блоками по 20
- **Учить:** тянет активные слова (фильтр по части речи), даёт упражнения по недобранным типам, считает
  блок из 20 раундов → сводка → «ещё блок» / в меню. Выбор слова — взвешенный (больше недобрано → чаще),
  guard от повтора подряд. Этап-2 слова, отработанные сегодня, временно выпадают (дневной лимит).
- **Проверить выученные:** отдельная кнопка; гоняет «Выученные»; при ошибке — «Забыл → вернуть в обучение».

## 8. Несколько значений (senses[])
Одно слово = один Item с массивом значений. В ознакомлении — все значения; в «Переводе» засчитывается любое
верное и периодически проверяются вторичные смыслы (even: даже / чётный / ровный). Часть речи может
отличаться по значениям (multi-POS).

## 9. Модель данных
```
Item:     id, lemma, pos, freq_rank, ipa, is_expression?, is_irregular?,
          forms?{past,pp,pp_alt}, senses[]{gloss, pos, example}, usage?
Progress: { list:'catalog|active|solid|learned', stage:1|2,
            points:{listen,translate,context,write,speak,forms},
            day:{type->YYYY-MM-DD}, passive }
```

## Ключевые решения (v2, зафиксировано)
- Платформа PWA, хранилище localStorage, офлайн; TTS/ASR — Web Speech, fallback самооценка + «Пропустить».
- Слова через **пополнение** вручную (каталог), не авто-влёт. Тренировка/тест — блоками по 20.
- 4 списка: каталог / активные / железно знаю / выучено (два последних — раздельные, с возвратом).
- Баллы по типам: +1 / −2 (≥0). Этап 1 — 10/тип; Этап 2 — 30/тип, 1×/день/тип; пассивные → сразу Этап 2.
- Значения — `senses[]` в одном Item; перевод драит вторичные смыслы. Формы — для irregular verbs.
- База 15k лемм по частоте; контент (senses/примеры) — AI-генерация траншами.

## Как запустить (MVP)
```
cd "English dictionary/app"
python3 serve.py        # http://localhost:8123
```
Открыть http://localhost:8123. На десктопе работает всё, включая микрофон. На iPhone — тот же адрес в
браузере (произношение — самооценка/пропуск).
